# API list

## User
localhost:3000/api

## /user
查询所有用户  get('/') 
根据userName查询用户  get('/:userName')
新增用户  post('/add')
登录  post('/login')
退出  get('/signout')
根据ID更新用户信息  post('/:id')
根据ID删除用户信息  delete('/:id'
